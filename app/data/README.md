# 📂 data/

Папка для всіх даних та логів програми. **Ніколи не комітити в Git** (окрім цього README).

## Що сюди потрапляє

| Файл | Опис | Створюється |
|---|---|---|
| `netguardian.db` | Основна БД (пристрої, події) | автоматично при першому запуску |
| `netguardian_data.db` | БД моніторингу пінгу/статистики | автоматично |
| `lan_trust.db` | Довірені пристрої в LAN | автоматично |
| `ping_log.json` | Історія пінгу (прогноз) | автоматично |

## Міграція зі старої структури

Якщо у тебе вже є ці файли в корені старого проекту — просто скопіюй їх сюди перед першим запуском, щоб зберегти історію даних. Інакше програма створить нові порожні БД.

```bash
# З кореня старого проекту:
mv netguardian.db data/
mv netguardian_data.db data/
mv lan_trust.db data/
mv ping_log.json data/
```
